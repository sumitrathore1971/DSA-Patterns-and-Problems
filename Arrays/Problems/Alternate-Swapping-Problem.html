Alternate Swapping

Problem Statement

Given an array of integers, perform alternate swapping of its elements. That means, swap the 0th and 1st element, 2nd and 3rd element, and so on. If the array has an odd number of elements, the last element remains as it is.

Example:

Input:  arr = [1, 2, 3, 4, 5, 6]
Output: [2, 1, 4, 3, 6, 5]

Input:  arr = [10, 20, 30, 40, 50]
Output: [20, 10, 40, 30, 50]

Approach

Traverse the array with a step of 2. For each pair of consecutive elements, swap them. If the last element has no pair (in case of odd-sized arrays), leave it as it is.

Time Complexity: O(n)

Space Complexity: O(1)
